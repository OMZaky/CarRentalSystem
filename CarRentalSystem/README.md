# Tripzy Car Rental System 🚗
An enterprise-level WinForms application with Azure SQL integration.

## 🛠 Tech Stack
- **Language:** C# (.NET 10)
- **Database:** Azure SQL (EF Core 10)
- **Architecture:** Repository Pattern / Code-First Migrations

## 🚀 Local Setup
1. Clone the repository.
2. Create an `appsettings.json` file in the project root (Copy from `appsettings.template.json`).
3. Enter the shared Azure SQL credentials in your local `appsettings.json`.
4. Run `Update-Database` in the Package Manager Console.
